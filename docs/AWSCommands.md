# Helpful AWS COMMANDS
## LIST TRUVEN FILES
``` sh
aws s3 ls s3://cdm-builder-setup-files/truven-files/  --region us-east-1
```
## LIST VOCAB FILES
``` sh
aws s3 ls s3://cdm-builder-setup-files/vocab-files-files/  --region us-east-1
```
## List functions
``` sh
aws lambda list-functions --region us-east-1
```
